## Instalação e Executando

 - Clone:
`https://github.com/PrJnr/RepositoriesReact-Native.git`

 - Instalando as Depêndencias:
`yarn` ou `npm install`

 - Executando o Projeto:
    -Android:
    `yarn react-native run-android`
    -iOS
    `yarn react-native run-ios`
